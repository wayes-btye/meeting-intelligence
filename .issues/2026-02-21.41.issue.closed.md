## Problem
Claude's answers in the Chat page render as plain text (`whitespace-pre-wrap`). Claude returns markdown — bold headers, bullet lists, numbered steps — which displays as raw symbols and is hard to read.

Relevant file: `frontend/app/chat/page.tsx:155`

## What to change

**Install packages:**
\`\`\`bash
cd frontend
npm install react-markdown remark-gfm
\`\`\`

**Replace plain-text paragraph with ReactMarkdown:**
\`\`\`tsx
import ReactMarkdown from 'react-markdown'
import remarkGfm from 'remark-gfm'

// Replace:
<p className="... whitespace-pre-wrap">{result.answer}</p>

// With:
<div className="prose prose-sm max-w-none dark:prose-invert">
  <ReactMarkdown remarkPlugins={[remarkGfm]}>{result.answer}</ReactMarkdown>
</div>
\`\`\`

**Install Tailwind typography plugin** (for `prose` classes):
\`\`\`bash
npm install @tailwindcss/typography
\`\`\`
Add to `tailwind.config.js`:
\`\`\`js
plugins: [require('@tailwindcss/typography')]
\`\`\`
