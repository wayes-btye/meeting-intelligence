## Problem
There is no way to delete a meeting from the UI. Cleaning up test uploads requires going into the Supabase dashboard directly.

## Backend — new endpoint

Add `DELETE /api/meetings/{meeting_id}` in `src/api/routes/meetings.py`:

\`\`\`python
@router.delete("/api/meetings/{meeting_id}", status_code=204)
async def delete_meeting(meeting_id: str) -> None:
    client = get_supabase_client()
    # Delete chunks first (or rely on CASCADE if FK is set up)
    client.table("chunks").delete().eq("meeting_id", meeting_id).execute()
    client.table("extracted_items").delete().eq("meeting_id", meeting_id).execute()
    client.table("meetings").delete().eq("id", meeting_id).execute()
\`\`\`

## Frontend

**`frontend/lib/api.ts`** — add helper:
\`\`\`typescript
export async function deleteMeeting(meetingId: string) {
  const res = await fetch(\`\${API_URL}/api/meetings/\${meetingId}\`, { method: 'DELETE' })
  if (!res.ok) throw new Error('Delete failed')
}
\`\`\`

**Meetings page** — add a trash icon button per row. Show a confirmation dialog before firing. Remove the row from local state on success.
