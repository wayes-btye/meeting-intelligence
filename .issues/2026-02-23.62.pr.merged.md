## What this adds

- POST /api/meetings/{id}/image-summary calls Gemini 3 Pro Image (Nano Banana Pro, gemini-3-pro-image-preview) to generate a visual summary infographic, with automatic fallback to gemini-2.0-flash-exp + response_modalities IMAGE config if the primary model ID is unavailable
- Returns meeting_id, image_data (base64), mime_type
- Graceful 501 if GOOGLE_API_KEY is not configured
- ImageSummaryResponse Pydantic model added to src/api/models.py
- google-generativeai>=0.8.0 added to pyproject.toml
- api.imageSummary(meetingId) helper added to frontend/lib/api.ts
- Generate Visual Summary button in meeting detail panel, only visible when meeting has a transcript
- Click triggers spinner then renders image inline; friendly error on 501

## Warning: manual testing required

The happy path calls the live Gemini API. Automated tests cover only 501, 404, and 400 paths.
